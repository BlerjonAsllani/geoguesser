# GeoGuesser (Projekt GEOG)

## Projektbeschreibung
Dieses Projekt ist eine vereinfachte GeoGuessr-ähnliche Webanwendung.
Spieler sehen ein Bild eines Ortes und müssen auf einer Karte schätzen,
wo sich dieser Ort befindet. Anhand der Distanz zwischen Tipp und
tatsächlicher Position wird das Ergebnis bewertet.

## Ziel
- Grundlegende Webentwicklung üben
- Arbeiten mit Karten, Koordinaten und Distanzen
